# my-locations

Application to store custom location points on the interactive map.

## Development

### Build & Run

```shell
cp .env.example .env.dev
make dev
```

### Stop & Remove

```shell
make dev-stop
```

## Production

### Build & Run

```shell
cp .env.example .env.prod
make prod
```

### Stop & Remove

```shell
make prod-stop
```
